# 🚀 AI Career Assistant

An AI-powered web application that analyzes resumes and provides personalized career insights.

---

## 🌟 Features

- 📄 Resume vs Job Description Matching (BERT-based NLP)
- 📊 Match Score & Skill Gap Analysis
- 🤖 AI-powered Career Suggestions
- 🎯 Recommended Job Roles
- 🔐 Login & Signup System
- 📄 Downloadable PDF Report
- 📈 Interactive Dashboard with Charts

---

## 🧠 Tech Stack

- Python
- Streamlit
- Sentence Transformers (BERT)
- Transformers (HuggingFace)
- Plotly
- ReportLab
